# Product Description

**SmartCatAI** is a plugin for the Unreal Engine 5.7.2 

**SmartCatAI** â€” an AI cat character that is the target character in a chase game. The **Cat** will be chased by **Baby**. The **Cat's** movements will be based on procedural IK. Primary development platform is macOS (Apple Silicon).

It is important to remember that the **Cat** and the **Baby** are not enemies but playmates.

## Cat Behaviors

- Run/walk animations
- Basic Jump
- Jump up to and down from a perch
- Head turn to face **Baby**
- Yawn
- Wink
- Model style slow walk also meant for walking on narrow ledges
- Butt/tail shake
- Cuddle with **Baby**
- Escape location awareness including the location's difficulty level.
- Escape algorithm based on level and **Baby** location. Include a risk taking/ attitude element.
- Whisker twitching
- Know **Baby** location
- Hear **Baby**
- Knock objects out of the way.
- Behavior scripts for events like **Baby** sleeping, **Baby** getting hurt, getting caught, etcetera.

## Test Levels/Maps

Test level are meant for building an perfecting Cat Behaviors. The level should have a keyboard interface for launching various behaviors. The keyboard interface should only exist in a Test Level.

**Baby** will always be PlayerStart.
